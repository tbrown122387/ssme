Detailed Description

Class that performs multinomial resampling for the Liu-West filter. For justification, see page 244 of "Inference in Hidden Markov Models".

Liu West filter but without the auxiliary particle filter thing, and it allows the state transition to depend on covariates.

Liu West (version 2) filter future simulator add-on.

Liu West filter but without the auxiliary particle filter thing.
